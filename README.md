# basic-C

＄換鈔 (if else 練習) -> 輸入數字-> 計算出最少可換紙鈔或硬幣

＄數字轉正體中文字 (switch case 練習) -> 輸入數字 ex:1023 -> 印出：壹仟零貳拾參元

＄箭頭 (for 巢狀迴圈練習) -> 輸入箭頭層數，用＊印出對稱箭頭
